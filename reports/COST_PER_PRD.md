# Cost Per PRD

Metrics below represent the cost of generating a single PRD using the 7B model.

- **Total LLM Calls**: 11
- **Total Expected Input Tokens**: ~16,384 tokens
- **Total Expected Output Tokens**: ~7,800 tokens
- **Average PRD Generation Time**: ~80.5 minutes
- **Shortest Stage**: UX / Tech Critique (~4.73 minutes)
- **Longest Stage**: Lead PM Draft (~9.89 minutes)

*Cost in USD is $0.00 since inference is strictly local, but the temporal cost is 80.5 minutes of compute.*
